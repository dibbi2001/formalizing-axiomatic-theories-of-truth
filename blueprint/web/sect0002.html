2 Preliminaries

2.1 Syntax

The ordinary definition of a first-order language \(\mathcal{L}\) proceeds from the specification of a signature \(\mathcal{S}\) and then an inductive definition of the way formulas can be build with elements from \(\mathcal{S}\), the set of logical connectives, variables and the quantifiers. See that for any first order language all these elements except the signature are the same. Hence, we could think of a specific first order language as being fully specified by its signature. For reasons of conceptual compactness that is the approach [ 3 ] have used for the specification of a language.

Definition 1 first-order language
✓
#

Lean declarations

  • FirstOrder.Language

Let \(A\) be a set of function symbols and \(B\) a set of predicate symbols. Then, a first-order language is a structure \(\langle F, R \rangle \), where

  1. \(F : \mathbb {N} \to A\) and

  2. \(R : \mathbb {N} \to B\).

Note that this defines a language as consisting only of functions and relations, whereas traditionally a language also contains a set of constants. Observe, however, that constants can be modelled as \(0\)-ary functions, so this definition of a language does not limit expressive power. By providing concrete sets of function and relation symbols to \(F_i\) and \(R_i\) we obtain a specific language.

We will be implementing the languages \(\mathcal{L}\) and \(\mathcal{L}_T\) that are necessary for the proof of Halbach’s [ 1 ] Theorem 7.5 of the conservativity of TB and UTB over PA.

The language of peano arithmetic including syntactic functions and predicates is the first-order language \(\mathcal{L} = \langle F, R \rangle \), where

  1. \(F\) is defined by

    1. \(F(0) = \{ \texttt{null}\} \),

    2. \(F(1) = \{ \texttt{succ},\texttt{num}¸\texttt{neg},\texttt{forall},\texttt{exists},\texttt{denote}\} \),

    3. \(F(2) = \{ \texttt{add},\texttt{mult},\texttt{conj},\texttt{disj},\texttt{cond}\} \) and

    4. \(F(3) = \{ \texttt{subs}\} \) and

  2. \(R\) is defined by

  3. \(R(1) = \{ \texttt{Variable},\texttt{Constant},\texttt{Closed\_ Term},\texttt{Term},\texttt{Formula}_{\mathcal{L}}, \texttt{Sentence}_{\mathcal{L}},\texttt{Formula}_{\mathcal{L}_T},\texttt{Sentence}_{\mathcal{L}_T}\} \).

  • FirstOrder.Language.Term

Let \(\mathcal{L} = \langle F, R \rangle \) be a first-order language and \(\alpha \) a set used to index free variables. Then the set of terms with respect to \(\mathcal{L}\) and \(\alpha \) denoted \(\mathcal{T}(\mathcal{L},\alpha )\) is the smallest set such that

  1. the set of variables with respect to \(\alpha \) denoted \(\mathcal{V}(\alpha ) = \{ x_i | i \in \alpha \} \subseteq \mathcal{T}(\mathcal{L},\alpha )\) and

  2. for all \(i \in \mathbb {N}\), if \(f \in F(i)\) and \(x_1,...,x_i \in \mathcal{T}(\mathcal{L},\alpha )\), then \(f(x_1,...,x_i) \in \mathcal{T}(\mathcal{L},\alpha )\).

The concept of Term is used to define the concept BoundedFormula.

Definition 5 BoundedFormula

Uses

Lean declarations

  • FirstOrder.Language.BoundedFormula

Let \(\mathcal{L} = \langle F, R \rangle \) be a first-order language, \(\alpha \) a set indexing free variables, \(n\) the intended number of variables bound by a quantifier and \(\mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\) a set of terms. Then, the set of bounded formulas with respect to these variables \(\mathcal{B}(\mathcal{L},\alpha ,n)\) is the smallest set such that

  1. \(\bot \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  2. if \(t_1,t_2 \in \mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\), then \(t_1 = t_2 \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  3. for all \(i \in \mathbb {N}\), if \(P \in R(i)\) and \(t_1,...,t_i \in \mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\), then \(P(t_1,...,t_i) \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  4. if \(f_1,f_2 \in \mathcal{B}(\mathcal{L},\alpha ,n)\), then \((f_1 \Rightarrow f_2) \in \mathcal{B}(\mathcal{L},\alpha ,n)\) and

  5. if \(f \in \mathcal{B}(\mathcal{L},\alpha ,n+1)\), then \(\forall f \in \mathcal{B}(\mathcal{L},\alpha ,n)\).

A Formula is defined as a BoundedFormula that has no variables that still need to be bound.

A Sentence is then defined as a Formula that has no free variables.
