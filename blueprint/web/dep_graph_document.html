Definition 5BoundedFormula

Let \(\mathcal{L} = \langle F, R \rangle \) be a first-order language, \(\alpha \) a set indexing free variables, \(n\) the intended number of variables bound by a quantifier and \(\mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\) a set of terms. Then, the set of bounded formulas with respect to these variables \(\mathcal{B}(\mathcal{L},\alpha ,n)\) is the smallest set such that

  1. \(\bot \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  2. if \(t_1,t_2 \in \mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\), then \(t_1 = t_2 \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  3. for all \(i \in \mathbb {N}\), if \(P \in R(i)\) and \(t_1,...,t_i \in \mathcal{T}(\mathcal{L}, \alpha \cup \{ x_1,...,x_n\} )\), then \(P(t_1,...,t_i) \in \mathcal{B}(\mathcal{L},\alpha ,n)\),

  4. if \(f_1,f_2 \in \mathcal{B}(\mathcal{L},\alpha ,n)\), then \((f_1 \Rightarrow f_2) \in \mathcal{B}(\mathcal{L},\alpha ,n)\) and

  5. if \(f \in \mathcal{B}(\mathcal{L},\alpha ,n+1)\), then \(\forall f \in \mathcal{B}(\mathcal{L},\alpha ,n)\).

Definition 1first-order language

Let \(A\) be a set of function symbols and \(B\) a set of predicate symbols. Then, a first-order language is a structure \(\langle F, R \rangle \), where

  1. \(F : \mathbb {N} \to A\) and

  2. \(R : \mathbb {N} \to B\).

Definition 4Term

Let \(\mathcal{L} = \langle F, R \rangle \) be a first-order language and \(\alpha \) a set used to index free variables. Then the set of terms with respect to \(\mathcal{L}\) and \(\alpha \) denoted \(\mathcal{T}(\mathcal{L},\alpha )\) is the smallest set such that

  1. the set of variables with respect to \(\alpha \) denoted \(\mathcal{V}(\alpha ) = \{ x_i | i \in \alpha \} \subseteq \mathcal{T}(\mathcal{L},\alpha )\) and

  2. for all \(i \in \mathbb {N}\), if \(f \in F(i)\) and \(x_1,...,x_i \in \mathcal{T}(\mathcal{L},\alpha )\), then \(f(x_1,...,x_i) \in \mathcal{T}(\mathcal{L},\alpha )\).
